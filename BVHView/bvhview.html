Instructions

1. Prepare SMPL-X mesh

  1. Visit https://smpl-x.is.tue.mpg.de/, register, and log in
  2. Download SMPL-X Blender Add-on from Download page
  3. Open Blender
  4. In Edit > Preferences > Add-ons > top-right arrow > Install from disk... and select the add-on file
  5. Click File > New > General
  6. Delete all objects in scene
  7. Open the right panel and select SMPL-X
  8. Choose Neutral model and press Add
  9. Export as glTF
    • Click File > Export > glTF 2.0
    • Operator Presets > Restore Operator Defaults
    • Set Format > glTF Separate
    • Enable Data > Armature > Add Leaf Bones, disable Animation
    • Name it (e.g. mesh.gltf) > click Export glTF 2.0
  10. Close Blender

2. Convert BEAT2 .npz to .bvh

  1. Setup GENEALeaderboard/smpl2bvh
  2. python smpl2bvh.py --gender NEUTRAL --poses "[...]/1_wayne_0_2_2.npz" --output "[...]/output.bvh"

3. Visualize BEAT2 data

  1. Drag-drop mesh .bin and .gltf into BVHView
  2. After mesh loads, drag-drop the .bvh file
  3. Disable Draw Capsules

    4. Note: To load a new .bvh, click Clear, then drag-drop again.
