BVHView Instructions

How to use with SMPLX:

  • Download the SMPLX .fbx meshes from https://smpl-x.is.tue.mpg.de/
  • Follow these instructions to convert the FBX mesh to GLTF: https://github.com/TeoNikolov/BVHView?tab=readme-ov-file#custom-meshes
  • Drag-drop the .bin, .gltf, .png files into BVHView
  • Wait until the mesh has loaded (a character will appear)
  • Drag-drop a .bvh file
  • Turn off "Draw Capsules"
  • To load new .bvh files, press "Clear" on the left, then drag-drop your new .bvh

Convert BEAT2 .npz to .bvh:

  • Setup smpl2bvh from this fork: https://github.com/GENEALeaderboard/smpl2bvh
  • Use command *python smpl2bvh.py --gender NEUTRAL --poses "[...]/1_wayne_0_2_2.npz" --output "[...]/output.bvh"*
  • Drag-drop .bvh into BVHView. Make sure to use the same SMPL-X mesh as the one in the previous step
